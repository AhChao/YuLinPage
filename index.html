作品集

我用程式將想法具現化。或者是遊戲輔助工具，或者是構思的遊戲點子，或者是研究相關的專題。

  • 2017

    Rainbow Puzzle

    Game Design - Web - JS

    自己設計的小遊戲，靈感來自「全字母句」─要讓一個句子充滿所有字母以測試打字。想到了這個利用彩虹對對碰的遊戲，讓玩家挑戰用最少的次數打完手牌。

  • 2017

    Lunarrator

    Classwork - IOS - ObjectiveC

    修習系上課程的IOS期末作品，使用一般月曆結合農民曆的節氣與吉凶宜忌概念，讓記事的月曆更加多樣化，也具有基本記事、紀錄活動、搜尋好日子等功能。

  • 2017

    Royal Banquet

    Game Design - Web - JS

    靈光一閃後構思的遊戲，主題是國王舉辦了個盛大宴會，要讓大家來參加，玩家扮演的是帶人入座的門衛，把人帶到他們想接觸的對象身邊就能獲得分數，嘗試在每關獲得高分吧。

  • 2017

    UnlimitedBallGame

    Game Design - Web - JS

    嘗試結合SVG與JS在網頁上寫的動態闖關遊戲，設計各式各樣的方塊，玩家操作螢幕中的小球，一次又一次的闖關以達到紅色過關點！備有儲存Replay與和影子(過去的自己)競賽等功能。

  • 2018

    For The Winter

    Game Design - Web - JS

    發想簡單操作的遊戲，結合運氣與策略，玩家扮演的是過冬的松鼠，趕緊在冬天到來前儲備足夠的食糧吧！也別忘了多開拓更多拿取食糧的管道，只靠著基本的方法可是找不到松果的喔！

  • 2018

    Sdorica Filter

    Game Tool - Web - JS

    由雷亞遊戲股份公司發行的Sdorica是一款操縱角色進行闖關，消除方塊發動技能的獨特手機遊戲。遊戲的內角色眾多，各有特色相異的技能，為方便玩家在編組隊伍時更快找到需要的角色，於是寫了這個篩選器。

  • 2018

    BS Dictionary

    Game Tool - Web - JS

    BlackSurvival是由韓國遊戲公司ARCHBEARS所製作的一款BattleRoyal類型手機遊戲。這款遊戲以類即時回合制的方式透過探索與合成來呈現BR主題，這個工具具中英日三種語言，幫助玩家規劃遊戲中的套路，更容易贏取勝利。

  • 2019

    DotaChessBoard

    Game Tool - Web - JS

    Dota自走棋可以說是2019首款爆紅遊戲，基於Dota自製地圖上的基礎發展出的8人策略遊戲，受到全球玩家的歡迎。遊戲內不同種族與職業會發動增益效果，為方便玩家搜尋自己所缺的棋子和對自己的陣容現況了解，誕生了這個工具。
